# Install

## Prerequisites

- Ruby
- Sass
- Node

## Install
```shel
gem install bourbon
gem install neat
sudo npm install -g gulp
sudo npm install -g bower
```

## Download Dependencies

```
cd ./front-end-redesign/
npm install
bower install
```

## Add Bourbon Files

```
cd style/plugins
bourbon install
neat install
```

## Run Server

```
gulp init
gulp server
```

## Open Browser

`http://localhost:8080/`
