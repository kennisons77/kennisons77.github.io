Icons

The icon of the project are contain inside an svg sprite, individual icons are inserted into the page using the use tag with the xlink:href attribute, which reference the file url and the id of the icon, this is desirable, given that is makes proper use of the browser cache. To add the default style to an svg tag, the class .icon needs to be added or extended.

Important

The xlink:href attribute is not supported by IE 10, 9 and EDGE, but a polyfill is available to deal with this issue svg4everybody.

Basic Usage

              
// scss
#fansy-icon{
  
  @extend .icon;
}

// html
<svg id="fansy-icon" viewBox="0 0 48 48" class="icon icon-ligth">
  <use xlink:href="img/sprite.svg#icon-menu">
</svg>
